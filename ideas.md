## 1. 赛博朋克霓虹风格 (Cyberpunk Neon)
**设计运动**: Cyberpunk / Synthwave
**核心原则**:
1. **高对比度**: 深色背景搭配高饱和度霓虹色，营造强烈的视觉冲击。
2. **科技感**: 使用网格、故障艺术(Glitch Art)、数据流等元素。
3. **动感**: 强调光影流动和动态效果，模拟电子信号传输。
4. **未来主义**: 字体和图标设计偏向几何化和数字化。

**色彩哲学**:
- 背景: 深蓝黑 (#050A14) 或 深紫黑 (#0D0221)
- 主色: 霓虹粉 (#FF00FF), 激光蓝 (#00FFFF), 电光紫 (#BF00FF)
- 意图: 营造一种充满激情、未来感和不确定性的氛围，非常适合抽奖这种刺激的场景。

**布局范式**:
- **非对称分割**: 使用倾斜线条或不规则几何图形分割区域。
- **悬浮层级**: 关键元素（如抽奖按钮、中奖名单）采用悬浮卡片设计，带有发光边缘。

**标志性元素**:
- **霓虹边框**: 按钮和卡片带有脉冲式的霓虹光晕。
- **故障文字**: 标题或重要信息偶尔出现轻微的故障抖动效果。
- **网格背景**: 动态滚动的透视网格背景。

**交互哲学**:
- **即时反馈**: 任何操作都伴随着光效或声音反馈（如果支持）。
- **高能触发**: 抽奖按钮按下时有明显的能量积蓄和释放动画。

**动画**:
- **抽奖过程**: 数字流快速滚动，名字像代码解密一样逐个显现。
- **中奖展示**: 名字出现时伴随爆炸式的粒子效果或强光闪烁。

**排版系统**:
- 标题: **Orbitron** 或 **Russo One** (宽体、几何感强的无衬线字体)
- 正文: **Rajdhani** 或 **Roboto Mono** (带有科技感的等宽或半等宽字体)

## 2. 奢华黑金盖茨比风格 (Luxury Art Deco)
**设计运动**: Art Deco / Great Gatsby
**核心原则**:
1. **几何对称**: 强调线条的重复、对称和几何图形的组合。
2. **材质感**: 模拟金属（金、铜）、大理石、丝绒等高级材质的质感。
3. **优雅**: 字体和布局追求极致的平衡和优雅。
4. **仪式感**: 每一个交互都像是一场精心编排的仪式。

**色彩哲学**:
- 背景: 丝绒黑 (#1A1A1A) 或 深邃蓝 (#0F172A)
- 主色: 香槟金 (#D4AF37), 玫瑰金 (#B76E79)
- 意图: 营造高端、尊贵、隆重的年会氛围，让每一个奖项都显得价值连城。

**布局范式**:
- **中心对称**: 关键元素居中，周围装饰以放射状线条或边框。
- **画框式结构**: 屏幕边缘使用复杂的Art Deco纹样装饰。

**标志性元素**:
- **放射线条**: 类似太阳光芒的放射状线条装饰。
- **金属边框**: 带有光泽感的双线或三线金属边框。
- **菱形/扇形**: 经典的Art Deco几何图案。

**交互哲学**:
- **优雅过渡**: 页面切换和元素出现都采用缓慢而流畅的淡入淡出或滑动。
- **质感反馈**: 按钮点击有类似物理按键的下沉感和金属光泽变化。

**动画**:
- **抽奖过程**: 类似老式老虎机或胶卷滚动的机械质感动画。
- **中奖展示**: 名字在金色光芒中缓缓浮现，伴随流光扫过。

**排版系统**:
- 标题: **Playfair Display** 或 **Cinzel** (高对比度的衬线字体)
- 正文: **Montserrat** (简洁的几何无衬线字体，与标题形成对比)
</text>

## 3. 孟菲斯波普风格 (Memphis Pop)
**设计运动**: Memphis Design / Pop Art
**核心原则**:
1. **大胆撞色**: 使用高饱和度的对比色块，打破常规配色规则。
2. **几何图形**: 大量使用圆形、三角形、波浪线等基础几何图形进行装饰。
3. **趣味性**: 强调设计的游戏感和幽默感，不拘一格。
4. **扁平化**: 去除阴影和质感，强调图形本身的张力。

**色彩哲学**:
- 背景: 米白 (#FFF8E7) 或 浅灰 (#F0F0F0)
- 主色: 柠檬黄 (#FFD700), 珊瑚红 (#FF6B6B), 钴蓝 (#0047AB), 薄荷绿 (#98FF98)
- 意图: 营造轻松、活泼、充满创意的年会氛围，适合年轻化团队。

**布局范式**:
- **拼贴式**: 元素像贴纸一样随意散落在画面中，打破网格限制。
- **重叠堆积**: 图形和文字之间相互重叠，制造视觉层次。

**标志性元素**:
- **波浪线/斑点**: 作为背景纹理或装饰线条。
- **黑粗边框**: 图形和文字带有明显的黑色描边。
- **立体几何**: 简单的3D几何体漂浮在画面中。

**交互哲学**:
- **弹跳反馈**: 按钮和元素在交互时有明显的弹跳(Bouncing)效果。
- **惊喜感**: 鼠标悬停或点击时出现意想不到的图形变化。

**动画**:
- **抽奖过程**: 名字在不同颜色的几何图形中快速切换跳动。
- **中奖展示**: 名字像漫画弹幕一样弹出，伴随彩带喷射效果。

**排版系统**:
- 标题: **Abril Fatface** 或 **Fredoka One** (圆润或夸张的粗体)
- 正文: **Quicksand** 或 **Nunito** (圆润亲和的无衬线字体)
</text>
